Project Overview
The Smart E-Commerce Store is a Python-based console application that simulates a
modern online shopping platform. It manages a diverse inventory across categories like
Cosmetics, Electronics, and Clothing.
The main highlight of this project is the AI Shopping Assistant, which uses
collections.Counter to analyze customer purchase history and provide personalized product
recommendations based on preferred brands.

Features
⊛ Customer Management: New user registration and login using unique Customer IDs
⊛ Dynamic Inventory: 50+ products with category, brand, price, and stock management
⊛ Smart Shopping Cart: Browse by category or filter by brand
⊛ AI Recommendation Engine: Suggests products based on user purchase behavior
⊛ Purchase History: Track past orders and preferences
⊛ Input Validation: Error handling using try-except blocks

Technical Stack
Language: Python 3.13
Libraries Used: pandas, collections.Counter
Data Structures: Dictionaries and Lists

How to Use
1. Run the program and register/login
2. Browse products by category
3. Add or remove items from the cart
4. Checkout using Product IDs
5. View purchase history and AI recommendations
   
Project Logic
Products stored as: {Product ID: [Name, Brand, Price, Stock]}
Purchase history mapped with Customer IDs
airecommend() function identifies the most frequent brand and suggests products
